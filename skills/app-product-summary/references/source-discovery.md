# Source Discovery

初見のアプリコードベースで、`何のアプリか` と `どの体験が中心か` を読むための最初の入口。

## 最初に見るファイル

- `App.tsx`
- `main.tsx`
- `app/layout.tsx`
- `main.dart`
- `lib/main.dart`
- `*App.swift`
- `AndroidManifest.xml`
- `Info.plist`

ここでは次を見る:

- アプリ名
- 初期画面
- タブや主要導線
- onboarding / auth / paywall の有無

## 読み方

- 画面一覧から `何が主役の行動か` を先に当てる
- onboarding や strings から `どんな価値を約束しているか` を拾う
- settings / paywall / notification から継続利用の方向性を読む
