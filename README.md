# OpenSoloMakersKits

Reusable OSS-publishable skill bundles synced from `AgentOrchestration`.

## Layout

- `skills/`: public skill bundles

## Current Skills

- `skills/gen-feed-posts/`: app-aware feed post generation workflow bundle

## Local Source Of Truth

Each published skill keeps its editing source inside:

- `../AgentOrchestration/`

Treat `AgentOrchestration` as the source of truth.  
Use this repository as the OSS publishing destination only.

## Before Publishing

- Review content for internal-only references
- Choose and add a `LICENSE`
- Commit and push to GitHub repository `agentsync/OpenSoloMakersKits`
